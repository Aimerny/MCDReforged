Command
=======

Command is the most common way for users to interact with MCDR.
MCDR consider all console input and player chat messages from the server as user inputs.
For them, MCDR will try to parse the input as a command

Commands can be registered by MCDR itself and by plugins.
This page only introduces about commands from MCDR.
For the way to register command for MCDR plugin, check :doc:`here </plugin_dev/command>`
